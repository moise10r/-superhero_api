### Setup

```bash
git clone git@github.com:moise10r/superhero.git
cd ./superhero/client
```

## Getting Started

First, run the development server:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

To run locally

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Run first the server before running the client.
