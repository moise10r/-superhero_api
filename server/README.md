## Description

Superheroes Api

## Built With

- Nest

### Setup

```bash
git clone git@github.com:moise10r/superhero.git
cd ./superhero/server
```

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Running the app in the docker container

```bash
# Run
$ docker build -t superhero .

# And then
$ docker run -p 3000:3000 superhero

```

You can open it from the PORT : 3000

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
